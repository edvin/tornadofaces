Date Picker

#{docsController.tag.description}

Pickaday uses moment.js for client side date formatting. Be aware that client and server side formats may differ.

<t:datepicker id="mypicker" format="YYYY-MM-DD" value="#{docsController.expr('controller.date')}"> <f:convertDateTime pattern="yyyy-MM-dd"/> </t:datepicker>

More properties are available, see pickaday.js site.

Form Element

To lay out form components we recommend you use the pattern supported by the Foundation For Apps Forms documentation. Foundation for Apps has a great grid, and lots of sensible defaults for inputs.

However, to reduce noise and keep your code DRY, the form-element tag is intended to be a helper.

Form Element is a work in progress, for now we recommend laying out forms the vanilla Foundation for Apps way! 

<h:form>
    <t:grid-block>
        <t:form-element medium="6">
            <h:inputText pt:placeholder="Name"/>
        </t:form-element>
        <t:form-element label="Age" medium="6">
            <t:slider min="0" max="100" lower="20" connect="lower"/>
        </t:form-element>
    </t:grid-block>
</h:form>

To fully understand the markup for form elements, please consult the Foundation for Apps Documentation.

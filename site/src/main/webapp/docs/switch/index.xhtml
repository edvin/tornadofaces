Switch

#{docsController.tag.description}

There is a client side Widget API as well as AJAX server callbacks available.

<t:switch value="#{docsController.expr('bean.booleanValue')}"> <f:ajax listener="#{docsController.expr('controller.onSwitch')}" render="noticiation"/> </t:switch> <button onclick="TW('myswitch').toggle()">Toggle with Widget API</button>
Client Side Callbacks

onChange is called as the switch is flipped.

function onChange(switch) { // }
Ajax Behavior Events

change will be called when a switch is flipped

<t:switch> <f:ajax event="change" listener="#{docsController.expr('controller.onChange')}"> </t:switch> public void onChange(SwitchEvent event) { Switch _switch = event.getSwitch(); }

The listener will be called with an instance of io.tornadofaces.event.SwitchEvent that contains a reference to the switch.

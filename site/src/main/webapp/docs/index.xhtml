The Modern JSF Component Framework

TornadoFaces uses modern, stylable, responsive markup and CSS, based on Foundation for Apps. This gives you as a developer some serious power when it comes to customizing your applications look and behavior.

There is no dependency on AngularJS, but instead components are Widgets, defined with a clean and simple JavaScript API. Widgets require JQuery. You can disable the included version and use your own if you need to.

Standing on the shoulders of giants

TornadoFaces includes ideas, principles and occasionally some code from other great open source projects, including:

Foundation for Apps, PrimeFaces, Ladda, Pickaday, noUiSlider

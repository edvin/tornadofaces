TornadoFaces Introduction

The Modern JSF Component Framework

TornadoFaces uses modern, stylable, responsive markup and CSS and a sleek Widget API. We want every TornadoFaces application to look unique, so we made sure you can easily customize the look of every detail by extending our SASS styles.

Components have a clean and simple JavaScript Widget API written in JQuery. You can disable the included JQuery version and use your own if you need to.

Standing on the shoulders of giants

TornadoFaces includes ideas, principles and code from other great open source projects, including:

Foundation for Apps, PrimeFaces, Ladda, Pickaday, noUiSlider, Spectrum, Prism

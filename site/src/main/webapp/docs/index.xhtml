The Modern JSF Component Framework

TornadoFaces uses modern, stylable, responsive markup and CSS. Using SCSS you create your own custom stylesheet instead of adding overrides to the default theme. You get access to all the SCSS variables we define and you can easily remove components you don't need in your compiled stylesheet. This gives you as a developer some serious power when it comes to customizing your applications look and behavior.

Components have a clean and simple JavaScript Widget API written in JQuery. You can disable the included JQuery version and use your own if you need to.

Standing on the shoulders of giants

TornadoFaces includes ideas, principles and code from other great open source projects, including:

Foundation for Apps, PrimeFaces, Ladda, Pickaday, noUiSlider, Spectrum

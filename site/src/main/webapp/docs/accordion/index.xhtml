Client Side Callbacks

onItemChange is called as a tab is activated and the content is loaded. The callback receives the item element representing the activated item. 

function itemChanged(item) {
    // item is the new active element
}
Ajax Behavior Events

itemChange will be called when an item is toggled. 

<t:accordion>
    <f:ajax event="itemChange" listener="#{docsController.expr('controller.onChange')}">
</t:accordion>
public void onChange(ItemChangeEvent event) {
    Accordion accordion = event.getAccordion();
    Tab activeTab = event.getTab();
}

The listener will be called with an instance of io.tornadofaces.component.accordion.ItemChangeEvent that contains a reference to the new active item and the accordion.

Client Side API

Widget: TornadoFaces.widget.Accordion

Method Params Return type Description
select(index) index: Index of the item to select boolean Activate or toggle the selected item. Returns true if the selected item was activated, false if it was closed.
